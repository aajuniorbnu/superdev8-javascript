# SuperDev8 Java

Repositório com exercícios e exemplos desenvolvidos em **Java** durante os estudos no curso **SuperDev8**.

O objetivo deste repositório é praticar conceitos fundamentais de programação, lógica e orientação a objetos utilizando a linguagem Java.

---

## 📚 Conteúdo do Repositório

Este projeto contém exemplos e exercícios sobre:

- Estruturas básicas da linguagem Java
- Variáveis e tipos de dados
- Operadores aritméticos e lógicos
- Estruturas condicionais (`if`, `else`, `switch`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Métodos e funções
- Orientação a Objetos
- Manipulação de listas e arrays
- Exercícios práticos de programação
